# Ado Hall of Worship — The Grail Movement

## Overview

Architectural design and visualization project for the Ado Hall of Worship belonging to The Grail Movement, Ado-Ekiti, Ekiti State.

The scope covers architectural redesign, modernization of the existing facility, landscape development, and site circulation planning. The project is currently at visualization and architectural design stage.

## Project Type

Religious Facility Renovation and Redevelopment

## Client

The Grail Movement

## Location

Ado-Ekiti, Ekiti State, Nigeria

## Scope

- Architectural redesign of the worship facility
- Landscape design and development
- Site circulation planning
- Exterior façade enhancement
- Architectural visualization
- Construction documentation

## Status

Ongoing — Visualization and Architectural Design Stage

## Deliverables

- Architectural Visualizations
- Construction Documentation
- Approval Drawings
- Landscape Design Documentation
