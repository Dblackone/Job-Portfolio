# 6-Bedroom Duplex, Ado — Construction & Interior

## Project Overview
A six-bedroom private residence in Ado, Ekiti State, delivered in two distinct phases: the structural construction of the building (foundation through roofing) and, once the shell was complete, a full interior design and fit-out package.

## Project Location
Ado, Ekiti State, Nigeria.

## Project Type

The project was delivered as two sequential phases. The structural phase followed standard site-engineering sequencing from foundation to roof. Once the building was structurally complete, the interior phase began: several design concepts were explored and discussed with the client, a preferred scheme was agreed, and a detailed interior package was developed and implemented to transform the finished shell into a cohesive, refined living environment.
