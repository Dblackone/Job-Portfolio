# Vollmann Architectural Portfolio

A 3D creator-style landing page built with React, TypeScript, Tailwind CSS, Framer Motion, and Lucide React, styled with the Dynamic Neutral design system (cream background, slate grey and gradient accents) and the Kanit typeface.

## Sections

1. **Hero** - massive gradient headline, magnetic mouse-following portrait, navbar, contact CTA
2. **Marquee** - dual-row scroll-driven image marquee
3. **About** - character-by-character scroll-reveal bio text
4. **Services** - Architectural Design, BIM Modeling & Coordination, Construction Project Management, 3D Visualization & Rendering, Interior & Landscape Design
5. **Projects** - sticky stacking project cards with scale-on-scroll effect

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
