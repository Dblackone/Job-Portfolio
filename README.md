# Akarakiri Portfolio System

## Overview

The Akarakiri Portfolio System is a structured portfolio development framework designed to showcase the professional works, projects, and expertise of Vollmann Olamide Akarakiri across multiple disciplines within the Architecture, Engineering, Construction (AEC), and Built Environment industries.

This repository serves as the central hub for creating, organizing, maintaining, and publishing professional portfolios tailored to different career opportunities, clients, consulting engagements, and project categories.

The system is designed to generate multiple specialized portfolios, including:

* Architectural Design Portfolio
* BIM & Revit Portfolio
* Interior Design Portfolio
* Landscape Architecture Portfolio
* Construction Management Portfolio
* Site Engineering Portfolio
* Design & Build Portfolio
* Project Management Portfolio
* Visualization & Rendering Portfolio
* Consultancy Portfolio

Each portfolio is organized independently while maintaining a unified professional identity and brand.
