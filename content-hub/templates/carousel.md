# Carousel Template (Instagram / LinkedIn)

> 3–8 slides, 1080×1350 (4:5). Slide 1 = the hook (must work alone). Last slide = CTA.
> Render slides with `graphics/engine/build_graphics.py` (type: `carousel`).

---

## Structure that works

| Slide | Role | Content |
|---|---|---|
| 1 | **Hook** | Big claim/number/question + kicker label. The "cover". |
| 2 | **Stakes** | Why this matters / the problem it solves. |
| 3–N | **Payload** | One step or point per slide. Number them. |
| N-1 | **Proof** | The real project / before-after / result. |
| N | **CTA** | "Save this · Follow @handle · DM for a model" + brand footer. |

## Writing rules
- Each slide = **one** idea, ≤ ~20 words of body. The image does the talking.
- Consistent kicker across slides (`BIM EXPLAINED · 03/06`).
- Slide 1 text must make sense with zero context (it's what shows in feed).
- Keep terracotta for emphasis only — one accent element per slide.

## Example skeleton — "5 Revit habits that save hours"
1. *5 Revit habits that save me hours every week* (hook)
2. *Habit 1 — Purge before every issue* (+ one line why)
3. *Habit 2 — View templates, not manual overrides*
4. *Habit 3 — Schedules to auto-quantify*
5. *Habit 4 — Worksets from day one*
6. *Habit 5 — Dynamo for the boring 200-sheet jobs*
7. *Save this. Follow @handle for one tip a day.* (CTA + footer)
